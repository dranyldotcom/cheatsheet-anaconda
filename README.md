# cheatsheet-anaconda

### to list all conda environments
```
conda env list
```
### to activate
```
conda activate
conda activate myenv
```
### to deactivate
```
conda deactivate
```
### to delete 
```
conda env remove -n myenv
conda env remove --name myenv
```
### to create
```
conda create -n myenv python=3.10
```
